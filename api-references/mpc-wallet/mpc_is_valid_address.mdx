---
title: Is Valid Address
api: GET /v1/custody/mpc/is_valid_address/ 
description: "This endpoint checks if a given address is valid for a specified coin. It also supports memos for certain coins. The response is a boolean indicating whether the address is valid or not."
---

<Tip> FAQ：[How to verify whether a withdraw address is legitimate?](/faqs/address-management#how-to-verify-whether-a-withdrawal-address-is-legitimate)</Tip>

#### Request
<ParamField query="coin" type="String" required>Coin code</ParamField>
<ParamField query="address" type="String" required>address; please use ‘｜’ to separate the addresses if there’s also a memo</ParamField>

#### Response
<ResponseField name="success" type="bool">request successful or failed</ResponseField>
<ResponseField name="result" type="bool">Whether the address is valid or not.</ResponseField>
