## Overview

[Token Swap](/en/apps/token-swap/introduction)'s Bridge feature enables secure cross-chain transfers between the following tokens and chains:

## BTC bridging
BTC can be bridged between the following chains:
- Bitcoin
- Ethereum
<Note>Currently, only Custodial Wallets (Asset Wallets) support BTC bridging.</Note>

## USDC and USDT bridging
USDC and USDT can be bridged between the following chains:
- Ethereum
- Solana
- Tron
- Op Mainnet
- Arbitrum
- Base (USDC only)
- Polygon
- BSC Chain
- Avalanche

## ETH bridging
ETH can be bridged between the following chains:
- Ethereum
- Op Mainnet
- Arbitrum
- Base
- BSC Chain
