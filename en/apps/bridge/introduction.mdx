Token Bridge is a [Cobo Portal App](/en/portal/portal-apps/introduction) maintained by Cobo. It is a powerful tool that enables users to perform cross-chain transactions with their [Custodial Wallet](/en/portal/custodial-wallets/introduction) assets. Token Bridge provides a secure and efficient way to move assets between different blockchain networks and perform token conversions.

## Key features

- **Cross-chain transfers**: Move assets between different blockchain networks.
- **Token conversion**: Convert between native and wrapped tokens.
- **Transaction monitoring**: Track and verify transaction status.
- **Security**: [Cobo Guard](/en/guard/introduction.mdx) approval system for transaction security.
- **Transaction history**: View and export detailed transaction records.
