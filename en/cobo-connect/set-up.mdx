Follow the steps to install Cobo Connect and import Cobo Portal wallet addresses.

## Install Cobo Connect

1. On the [Cobo Connect](https://chromewebstore.google.com/detail/cobo-connect/imnmigiomjbgehplnfpajkgeaibljnik?pli=1) extension page, click **Add to Chrome.**
2. Read the information in the pop-up window and click **Add extension** if you agree to the terms.
3. Disable other wallet extensions as suggested, then click **Continue**.
4. Create and confirm the password for Cobo Connect, then click **Continue**.
5. Follow the on-screen instructions and pin the extension for easier access, then click **Continue**. 

## Import wallet addresses into Cobo Connect

<Note>Currently, you can import addresses of [MPC Wallets](/en/portal/mpc-wallets/introduction) created under [Organization-Controlled Vaults](/en/portal/mpc-wallets/ocw/create-mpc-vault), [Web3 Wallets](/en/portal/custodial-wallets/web3-wallets/set-up), and [Smart Contract Wallets](/en/portal/smart-contract-wallets/introduction). Cobo Connect supports only EVM networks, so ensure that the addresses you import are compatible with EVM networks.</Note>

1. After the Cobo Connect extension is installed, select the Cobo Portal environment that you want to use to import wallet addresses.

    <img src="/en/images/cobo-connect/select-env.png" className="screenshot_modal" alt="Select environment"/>

2. After selecting the environment and logging in, you will be redirected to the **MPC Wallets** page in your selected environment (for example, Cobo Portal production environment).
    - To import MPC Wallet addresses, select a vault to proceed. 
    - To import Smart Contract Wallet addresses, go to the Smart Contract Wallets page first.
    - To import Web3 Wallet addresses, go to the Custodial Wallets page and select the **Web3 Wallets** tab.
3. For MPC Wallets and Web3 Wallets, click <img src="/en/images/common/icon_cobo_connect.svg" className="icon" alt="Cobo Connect"></img> next to the wallet you want to import addresses from. For Smart Contract Wallets, click <img src="/en/images/common/icon_more.svg" className="icon" alt="The overflow icon"/>, then click <img src="/en/images/common/icon_cobo_connect.svg" className="icon" alt="Cobo Connect"></img>. In the pop-up window, click **Allow** to confirm connection to Cobo Portal.

    <img src="/en/images/cobo-connect/location.png" className="screenshot_modal" alt="Click Cobo Connect icon"/>

4. Select the wallet addresses that you want to import, then click **Connect**.
5. On the confirmation page, click **Confirm**.
