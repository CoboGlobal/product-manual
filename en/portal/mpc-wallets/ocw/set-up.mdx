## Prerequisites

- You have set up your account and complete other required steps following the [Quick start guide](/en/portal/quick-start-guide-mpc).
- You have downloaded [Cobo Guard](/en/guard/download) and follow the on-screen instructions to set up Cobo Guard on your iOS devices.
- You have been assigned the [Operator](/en/portal/organization/roles-and-permissions) role or a role with equivelant permissions.

## Steps

Setting up an Organization-Controlled Wallet involve the below steps:
1. [Create an MPC vault](/en/portal/mpc-wallets/ocw/create-mpc-vault)
2. [Create private keys](/en/portal/mpc-wallets/ocw/holder-group-overview)
3. [Back up key shares](/en/portal/mpc-wallets/ocw/back-up-key-share-groups)
4. [Create an MPC Wallet](/en/portal/mpc-wallets/ocw/create-mpc-wallet)
