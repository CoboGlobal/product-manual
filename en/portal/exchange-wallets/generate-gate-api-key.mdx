When you link a Gate.io account to your Exchange Wallet, you will need to provide an API key and a secret key. Besides, you also need to include Cobo in the IP whitelist and assign specific permissions to the key.

This article explains how to generate an API key on Gate.io, including adding Cobo to the IP whitelist and setting permissions.

<Note>The content is subject to changes on Gate.io. For more details, refer to <a href="https://www.gate.io/help/guide/faq/17521/how-to-utilize-api">How to Utilize API</a>.</Note>

1. Log in to your Gate.io account.
2. Navigate to **My Profile** > **API Management**.
3. Select **Create API Key** to open a **Create API Key** page.
    <img src="/en/images/exchange-wallet/gate_1.png" className="screenshot_modal" alt="The Create API Key page" />
4. On the **Create API Key** page, enter a remark in the **API Key Remark** field.
5. In the **IP Permissions** section, select **Bind IP** and enter Cobo's IP address `54.201.57.0` in the text field.
6. In the **API Key Type** section, select **API v4 Key**.
7. In the **Account Type** section, select **Trading Account**.
8. In the **Permissions** section, enable the following options:
    - **Spot Trade**: **Read Only**
    - **Wallet**: **Read And Write**
    - **Withdraw**: **Read And Write**
    - **Subaccount**: **Read Only**
    - **Account**: **Read Only**
    - **Perpetual Futures**: **Read Only** 
    - **Delivery Futures**: **Read Only** 
    - **Option**: **Read Only** 
    <img src="/en/images/exchange-wallet/gate_2.png" className="screenshot_modal" alt="The permissions options" />

9. Click **Confirm** to generate the API key.

You have now generated and set up an API Key. You can go to Cobo Portal to link your Gate.io account with the API key and API secret key. To learn how to link your account, see [Set up Exchange Wallets](/en/portal/exchange-wallets/set-up).
