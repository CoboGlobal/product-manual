<Info>Due to the Bybit security incident, Safe Wallet has suspended third-party delegate call services. As a result, Cobo's Smart Contract Wallets and related services (including on-chain transaction policies) are temporarily unavailable. However, existing wallets and on-chain transaction policies can still be used. Recovery timeline will be announced later.</Info>

To enable seamless interaction with dApps using Smart Contract Wallets, you must first install Cobo Connect, a Chrome extension that functions as the gateway between Cobo Portal and the dApp ecosystem. Click here to learn more about [Cobo Connect](/en/cobo-connect/interact-with-dapps).

To install Cobo Connect, click into your Safe&#123;Wallet&#125; on Cobo Portal and click <img src="/en/images/smart-contract-wallet/cobo connect.svg" className="icon"></img> in the upper right corner. See the Cobo Connect [documentation](/en/cobo-connect/set-up) for the installation guide.
