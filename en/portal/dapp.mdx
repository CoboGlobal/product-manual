Currently, you can connect Cobo Wallets with dApps using the following two methods:

| Method                     | Reference Documentation                                                                                   | Supported Chains | Supported Wallet Types        |
| ------------------------ | -------------------------------------------------------------------------------------------------------- | ---------------- | ------------------------ |
| Cobo WaaS 2.0 API        | [Interact with dApps via API](https://www.cobo.com/developers/v2/api-references/transactions/call-smart-contract) | EVM, Solana      | Custodial wallets (Web3 Wallets), MPC wallets |
| Cobo Connect plugin  | [Interact with dApps via Cobo Connect](/en/cobo-connect/interact-with-dapps)        | EVM              | Custodial wallets (Web3 Wallets), MPC wallets |
