---
title: Get Wallet Addresses List
api: GET /v1/custody/web3_list_wallet_address/ 
description: "This endpoint returns a JSON response containing a list of wallet addresses for a given chain code. The response also includes the total count of wallet addresses for the given chain code. Pagination is supported through the use of page index and page length parameters."
---
#### Request 

<ParamField body="chain_code" type="String" required>Chain code for which to retrieve wallet addresses.</ParamField> 
<ParamField body="page_index" type="Int" required>Index of the page to retrieve.</ParamField> 
<ParamField body="page_length" type="Int" required>Number of wallet addresses to retrieve per page.</ParamField> 
<ParamField body="sort_flag" type="Int" >Address sorting method: If it is 1, it will be returned in a positive order of address creation time, or else it will be returned in a reverse order of address creation time.</ParamField>

#### Response 
<ResponseField name="total_count" type="Int" >Total number of addresses.</ResponseField>
<ResponseField name="addresses" type="String" >List of wallet addresses for the given chain code.</ResponseField>

<RequestExample>
```python Python
request(
    "GET",
    "/v1/custody/web3_list_wallet_address/",
```
