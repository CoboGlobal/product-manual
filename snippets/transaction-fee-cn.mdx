查看交易费用。费用将从您在第三步中选择的源钱包扣除，您需要确保源钱包中已开通相应的代币且有足够的余额支付。或者您可以启用 [Fee Station](/cn/portal/fee-station/introduction) 开通交易手续费代付。

根据交易速度，网络费用分为三个级别：慢、标准和快。要调整该费用，请点击下图圈出的按钮。所有字段都是必填项，并将根据您选择的交易速度自动填充。您可以根据自己的具体要求编辑各个参数。

- EVM 链：对于 EVM 链，EIP-1559 和 Legacy 方式的交易费用有所不同。
    - EIP-1559
        - 在 EIP-1559 方式下，用户可以定制 3 个参数：Max Priority Fee (gwei)、Max Fee 和 Gas Limit。
        - 预估费用的计算公式为 `Max Fee * Gas Limit`。
        - 当前支持的链为 Ethereum Mainnet（`ETH`），Sepolia Testnet（`SETH`）和 Scroll Sepolia Testnet（`SCROLL_ETH`）。

        <a id="eip1559"></a>	
        | 参数        | 定义                                                                                                                                                                                                                             |
        | --------         | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------  |
        | Max Priority Fee | Max Priority Fee 通常也被称为“矿工小费”，支付给矿工，激励他们优先处理您的交易。Max Priority Fee 是您愿意为交易支付的最高矿工小费。
        | Max Fee          | Max Fee 是您愿意为交易支付的 Gas 总价。                                                                                                                                                |
        | Gas Limit        | Gas Limit 是您愿意为执行交易或运行以太坊虚拟机 (EVM) 而支付的最高 Gas Unit 数量。不同的运行方式需要消耗不同数量的 Gas Unit。              |

    - Legacy
        - 在 Legacy 方式下，用户可以定义 2 个参数：Gas Price (gwei) 和 Gas Limit。
        - 预估费用的计算公式为 `Gas Price * Gas Limit`。

        | 参数 | 定义    |
        | -------- | -------------------------------------------------------------------------- |
        | Gas Price | Gas Price 是必须支付给验证者以便处理交易的 ETH（单位较小，称为 gwei）数量。 |
        | Gas Limit | Gas Limit 是您愿意为执行交易或运行以太坊虚拟机 (EVM) 而支付的最多 Gas Unit 数量。 |
- 基于 UTXO 的链
    - 用户可以自定义费率 (sat/vByte)。
    - 预估费用的计算公式为 `Fee Rate * Transaction Size`。

        | 参数 | 定义                                                                                                                               |
        | --------  | -----------------------------------------------------------------------------------------------------------------------------------------|
        | Fee Rate  | 费率的计算方式是您的交易在区块链上消耗的每单位数据对应多少 satoshis（缩写为 sats/vByte）。|

- Solana 链
    - 用户可自定义两个参数：Compute Unit（CU）Price 和 CU Limit。
    - 预估交易费用计算公式为：`Base Fee + (CU Price × CU Limit)`

        | 参数名        | 描述                                                                                                                                                      |
        |---------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
        | CU Price      | 每个计算单元的费用，用于提升交易优先级。单位为 Lamports。                                         |
        | CU Limit      | 交易允许消耗的最大计算单元（CU）数量。数值越高优先级越高，但费用也越高。                                                                                 |

    <Note>如需使用 Legacy 费用模型，请通过调用 API 接口并指定模型类型。</Note>

- Filecoin 链
    - 用户可自定义三个参数：Gas Fee Cap、Gas Premium 和 Gas Limit。
    - 预估交易费用计算公式为：`min(Gas Fee Cap, Base Fee + Gas Premium) × Gas Limit`

        | 参数名        | 描述                                                                                                                                                         |
        |---------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
        | Gas Fee Cap   | 每单位 Gas 可接受的最高费用，最终每单位 Gas 实际支付价格不会超过该值。                                                                                       |
        | Gas Premium   | 支付给区块生产者的优先费（Tip），用于提升交易打包优先级。                                                          |
        | Gas Limit     | 交易允许消耗的最大 Gas 数量。                                              |

    <Note>如需使用 Legacy 费用模型，请通过调用 API 接口并指定模型类型。</Note>
