OTC App 支持您通过持牌 OTC 服务商进行加密货币和法币的兑换，以及在服务商进行资产的充值、提取等管理功能。

<img src="/cn/images/portal-apps/otc/otc-in-app-store.png" className="screenshot_modal" alt="OTC作为Cobo Portal App显示"/>

## 主要特点

### 全面的 OTC 交易服务
灵活兑换您的各类资产（支持加密货币与加密货币、加密货币与法币、法币与法币之间的兑换），高效执行大额跨境资金转移。

### 一站式体验
在 Cobo Portal 内执行交易、查看交易历史和管理账户，无需跳转至第三方平台。

### 安全与合规
选择持牌且受监管的 OTC 服务商，共享您的 Cobo KYC 信息以加快开户流程；通过白名单地址确保加密货币转账安全。

## 使用简介

<Info>OTC 目前支持 dtcpay ([https://dtcpay.com/](https://dtcpay.com/)) 作为首个服务商，支持币种为 USDT、USDC、USD、SGD。未来将支持更多服务商。</Info>

### 1. 关联服务商账户
1. 在 OTC 中查看可用的服务商

   <img src="/en/images/portal-apps/otc/list.png" className="screenshot_full_screen" alt="OTC App 首页展示服务商列表"/>

2. 在选定的服务商处开设账户
3. 使用 API 凭证在 OTC App 中关联您的服务商账户

   <img src="/en/images/portal-apps/otc/activate.png" className="screenshot_full_screen" alt="激活服务窗口"/>

### 2. 配置转账设置
- 添加法币提现的银行账户
- 设置加密货币转账的地址白名单

### 3. 开始交易

<img src="/en/images/portal-apps/otc/trade.png" className="screenshot_full_screen" alt="资产兑换页面"/>

- 在加密货币、法币之间进行兑换
- 充值和提现法币
- 使用白名单内的 Cobo Portal 全托管钱包和 MPC 钱包地址进行加密货币转账
- 查看交易状态和历史记录

    <img src="/en/images/portal-apps/otc/history.png" className="screenshot_full_screen" alt="交易历史页面"/>

### 4. 管理账户
- 切换或解绑服务商账户
- 更新银行账户和地址白名单
- 获取服务商客服支持
- 更新账户连接的 API 凭证
