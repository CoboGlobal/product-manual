Earn 是由 Cobo 开发的 [Cobo Portal App](/cn/apps/introduction)。您可以通过它在 Cobo Portal 钱包中投资 DeFi 产品来获得收益。

<img src="/cn/images/portal-apps/earn/earn-in-app-store.png" className="screenshot_modal" alt="Earn作为Cobo Portal App显示."/>

## 支持的钱包、代币和理财产品

Earn 目前支持**全托管钱包**和 **MPC 钱包**，支持的代币和理财产品如下：

| 代币 | 理财产品 |
| ----- | -------------- |
| USDC (ETH 链) | Aave Core USDC Pool |
| USDT (ETH 链) | Aave Core USDT Pool |
| ETH | Aave Core ETH Pool |

## 主要特点

### 专业投资管理

提供多种风险收益特征的 DeFi 产品供您选择。经过精心筛选和整合的产品，可满足不同的投资策略。

### 便捷的投资操作

通过直观的界面轻松进行产品申购和赎回，清晰显示产品信息、投资状态和累计收益。

### 每日投资跟踪

通过全面的仪表板监控您的投资，显示总资产、历史收益和收益率。完全透明地了解您的投资仓位和交易历史。
