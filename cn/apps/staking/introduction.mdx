## 什么是质押？

质押是指将代币锁定，以支持区块链网络的安全和运行，从而获得奖励。Staking 支持不同的网络和资产。

## 主要特点

### 综合资产概览与监控

通过整合所有质押资产的视图、每种资产类型的详细信息以及实时更新的状态和奖励，全面掌控您的质押操作，保持质押仓位和活动历史的完全透明。

### 精简质押流程

通过直观的界面，轻松质押和管理您的资产，界面清晰显示质押状态，以及可质押、解除质押或赎回的资产。

### 可选择或自定义验证者

通过选择或自定义验证者来优化您的质押策略，以提升潜在奖励。

### 强大安全性

每一笔质押操作都受到 Cobo Portal 的用户角色检查机制和风险控制政策保护，并且提供专门支持以解答疑问。

### 质押 API

通过 Cobo WaaS 2.0 API 将质押集成到您的系统中，以增强对质押活动的控制及其灵活性。
