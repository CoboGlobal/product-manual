本文将指导您如何在 Token Swap 中使用兑换功能。

<Info>目前尚未支持跨链兑换。若要将资产跨链，请使用[跨链](/cn/apps/token-swap/bridge/introduction)功能。</Info>

## 前提条件
- 您已安装 Token Swap，如[浏览和安装 App](/cn/apps/browse-and-install)所示。
- 您已被分配[提币员](/cn/portal/risk-controls/roles-and-permissions)角色或具有等效权限的角色。
- 您正在从[全托管钱包](/cn/portal/custodial-wallets/introduction)或[ MPC 钱包](/cn/portal/mpc-wallets/introduction)兑换代币。
- 您已将涉及的钱包选为 Token Swap 的钱包范围，参见[浏览和安装 App](/cn/apps/browse-and-install) 中第 5 步。
- 您所兑换的代币属于支持的代币类型，参见[支持的代币与链](/cn/apps/token-swap/swap/supported-assets)。

## 步骤

1. 点击 Cobo Portal 中的 Apps 图标 <img src="/en/images/common/icon_apps.svg" className="icon" alt="Apps 图标" />，找到 Token Swap，然后点击**打开**。

2. 点击**兑换**标签以显示以下页面。

    <img src="/cn/images/portal-apps/swap/activity.png" className="screenshot_full_screen" alt="Swap 页面的活动表格" />

3. 点击**立即兑换**以打开**兑换**对话框。

    <img src="/cn/images/portal-apps/swap/now.png" className="screenshot_modal" alt="Image showing the Activity tab of the Token Swap app." />

4. 选择钱包，选择初始代币和目标代币，然后输入要兑换的金额。
<Note>如果您的钱包或代币未出现在列表中，请确保已将钱包选为 Token Swap 的钱包范围，并且代币属于支持的代币类型。</Note>
5. 查看报价、处理费用和预计时间。然后点击**预览**。
6. 确认详情并点击**提交**完成请求。
    <Note>一旦提交，兑换请求将无法取消。</Note>

您的兑换请求已提交。Cobo 将处理兑换请求，结算可能需要一些时间，具体取决于网络条件和可用流动性。您可以在 App 中的**活动**表格中查看兑换请求的状态。
