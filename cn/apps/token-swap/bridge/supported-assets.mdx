## 概述

[Token Swap](/cn/apps/token-swap/introduction) 的跨链（Bridge）功能支持以下代币与链之间的安全跨链转移：

## BTC 跨链
BTC 可在以下链之间进行跨链：
- Bitcoin
- Ethereum
<Note>目前仅支持通过全托管钱包（资产钱包）将 BTC 跨链。</Note>

## USDC、USDT 跨链
USDC、USDT 可在以下链之间进行跨链：
- Ethereum
- Solana
- Tron
- Op Mainnet
- Arbitrum
- Base （仅支持 USDC）
- Polygon
- BSC 链
- Avalanche

## ETH 跨链
ETH 可在以下链之间进行跨链：
- Ethereum
- Op Mainnet
- Arbitrum
- Base
- BSC 链
