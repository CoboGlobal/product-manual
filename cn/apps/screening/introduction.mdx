## 什么是 Screening？

Screening 是一款 [Cobo Portal App](/cn/apps/introduction)，通过自动化交易扫描简化反洗钱（AML）和反恐融资（CFT）合规流程。利用先进的“了解您的地址”（KYA）和“了解您的交易”（KYT）功能，它为您选择的 [MPC 钱包](/cn/portal/mpc-wallets/introduction)和[Web3 钱包](/cn/portal/custodial-wallets/web3-wallets/set-up)提供全面保护，同时让您的团队专注于业务增长。

一旦[安装](/cn/apps/screening/set-up)和配置，Screening 将持续实时扫描交易，提供智能风险扫描和自动化合规控制。

<Note>CipherOwl 可直接扫描交易，而使用 Elliptic 需要先进行 [API 密钥配置](/cn/apps/screening/screening-settings)。</Note>

## 主要特点

### 智能风险检测

Screening 与合规解决方案的领先提供商集成，利用先进的风险评估技术自动实时分析交易：

- [CipherOwl](https://www.cipherowl.com/)：Screening 安装后即可直接对交易进行扫描。
- [Elliptic](https://www.elliptic.co/)：使用前需要先进行 API 密钥配置，获取 API 密钥请联系 [Cobo 客服](mailto:help@cobo.com)。

Screening 分析交易后，会提供智能风险评估：
   - 使用 CipherOwl：交易根据 CipherOwl 的风险评分系统自动分为三个风险级别（低、中、高）。有关详细信息，请参阅 [CipherOwl 的地址风险评分文档](https://s.cipherowl.ai/Address-Risk-Score-1a0d90dd030f80b5904cd4aa76e15cea)。
   - 使用 Elliptic：风险以 0.0 到 10.0 的数值评分呈现（以 0.1 为增量），评分越高表示风险级别越高。有关详细信息，请参阅 [Elliptic 的风险评分文档](https://help.elliptic.co/en/articles/2641454-interpreting-risk-scores)。

   您可以根据交易的风险级别或评分来决定如何管理交易。

<Info>
当您[选定的扫描钱包](/cn/apps/screening/settings)中的交易被扫描时，Cobo 会将以下交易详情传输给您的 AML 服务商进行风险评估：
- CipherOwl：资产的链、发送地址（对于充币）、接收地址（对于提币）
- Elliptic：资产的链、接收地址（对于提币）、链上交易哈希（对于充币）
</Info>

### 自动防御和风险管理

Screening 分析交易后，会根据您配置的风险处置规则自动处理交易：

- [**风险处置规则**](/cn/apps/screening/risk-policies)：
  使用以下条件定义规则：
  - **交易方向**（转入/转出）
  - 风险（CipherOwl：低/中/高；Elliptic：0.0-10.0 的数值评分）
  - 金额范围（最小-最大，以美元计）

- **自动响应**：
  当规则匹配时，自动：
  - **批准**：允许交易继续进行
  - **批准并发出警报**：允许交易并发送风险交易通知
  - **拒绝**：
    - 对于转入交易：资金将被冻结
    - 对于转出交易：交易将被拦截
    - 始终发送风险交易通知

### 统一合规仪表板

通过直观的时间线视图高效管理所有合规活动，提供：

- 实时风险评估跟踪和审核状态
- 交易资金状态扫描
- 完整的审计跟踪和操作历史记录

集中式的时间线界面让您能够在同一页面轻松查看所有合规活动及其历史记录。
