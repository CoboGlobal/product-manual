<Note>
Screening App 将很快提供中文本地化版本。目前，本文档所有 UI 名称（如按钮名称）和术语均为 AI 翻译，可能与您在使用 App 时看到的英文 UI 不符。如需查看实际的英文 UI 名称，请参阅相应的英文手册：[Set up Screening](/en/apps/screening/set-up)
</Note>

Screening 是一款 [Cobo Portal App](/cn/apps/introduction)。要安装 Screening，请按照[浏览和安装 App](/cn/apps/browse-and-install) 中的说明操作。

<Note>在[安装](/cn/apps/browse-and-install)过程中，系统会要求您在**钱包范围**部分选择 App 可以访问的 [MPC 钱包](/cn/portal/mpc-wallets/introduction)和[Web3 钱包](/cn/portal/custodial-wallets/web3-wallets/set-up)。Screening 将自动扫描所选钱包的每笔交易。您可以按照[管理扫描钱包](/cn/apps/screening/settings)中的说明稍后修改这些钱包选择。</Note>

<Warning>
**重要提示**：安装后，请确保在使用 Screening 之前设置适当的[风险处置规则](/cn/apps/screening/risk-policies#风险处置规则)。如果没有配置任何规则，交易将在没有适当风险评估的情况下继续进行，这可能会使您的资产面临潜在风险。
</Warning>
