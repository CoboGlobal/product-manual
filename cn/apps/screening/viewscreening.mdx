### 时间线部分

时间线部分提供一个集中界面，在一个地方管理和跟踪所有合规活动：

- 实时跟踪审核状态变更
- 交易资金状态扫描
- 完整的审计跟踪和操作历史记录

#### 可用操作

根据交易的当前状态，可以使用以下操作：

| 操作 | 效果 |
| :---- | :---- |
| 批准 | • 对于入金：交易通过 AML 审核，用户可动用交易资金<br/>• 对于出金：交易通过 AML 审核，交易状态往下流转 |
| 批准并告警 | • 对于入金：交易通过 AML 审核，用户可动用交易资金，额外发送风险交易通知<br/>• 对于出金：交易通过 AML 审核，交易状态往下流转，额外发送风险交易通知 |
| 拒绝 | • 对于入金：交易不通过 AML 审核，交易资金被冻结<br/>• 对于出金：交易不通过 AML 审核，出金交易被拒绝 |
| 解除冻结 | • 对于入金：解除冻结的资金，用户可动用交易资金<br/>• 对于出金：无此操作 |
| 退回 | • 对于入金：交易资金从受益地址退回到用户指定地址<br/>• 对于出金：无此操作 |
| 隔离 | • 对于入金：交易资金从受益地址退回到用户指定的隔离地址<br/>• 对于出金：无此操作 |
