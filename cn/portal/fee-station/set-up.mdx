本文介绍如何在 Fee Station 中完成以下操作：

- 充值美元或稳定币
- 充值 Gas Token
- 设置手续费代付策略

## 充值美元或稳定币

Fee Station 支持充值美元法币或美元稳定币。充值后，您可以在 Fee Station 中使用这些资金来支付交易手续费、平台服务费和托管费等账单。

在**美元/美元稳定币**标签页，您可以查看当前账户中余额，充值记录及其支出记录，并发起充值操作。Fee Station 支持的稳定币类型请参考[支持充值的代币类型](/cn/portal/fee-station/introduction#支持充值的代币类型)。

1. 登录 [Cobo Portal](https://portal.cobo.com/login)。
2. 点击左下角的用户名，然后选择**团队** > **Fee Station**。
3. 在**美元/美元稳定币**标签页，在总价值右侧点击**充币**按钮，选择合适的充值方式，并按照指引完成充值。

    <img src="/cn/images/fee-station/deposit-u.png" className="screenshot_full_screen"/>

## 充值 Gas Token

您也可以直接为各条链的原生 Gas Token 充值，用于代付对应链上的交易手续费。

在**Gas Token**标签页，您可以查看当前账户中各链 Gas Token 的余额及其交易流水，并发起充值操作。Fee Station 支持充值的 Gas Token 请参考[支持充值的代币类型](/cn/portal/fee-station/introduction#支持充值的代币类型)。

1. 登录 [Cobo Portal](https://portal.cobo.com/login)。
2. 点击左下角的用户名，然后选择**团队** > **Fee Station**。
3. 切换到**Gas Token**标签页，在列表中找到目标链对应的 Gas Token（例如 TRX、ETH、MATIC 等）。
4. 在该 Gas Token 右侧点击 <img src="/cn/images/fee-station/deposit.svg" className="icon"></img> 充值图标，并按照提示完成充值操作。

    <img src="/cn/images/fee-station/deposit-gas-token.png" className="screenshot_full_screen"/>


## 设置手续费代付策略

在 **Gas Token > 付款偏好** 标签页，您可以为 MPC 钱包和 Web3 钱包基于支持的链分别设置链上手续费代付策略。

<img src="/cn/images/fee-station/preferences.png" className="screenshot_full_screen"/>

每条链的付款偏好决定了 Fee Station 何时为地址代付手续费：您可以选择仅在交易地址余额不足时代付，或始终代付。如果未为某条链配置付款偏好，则采用系统默认策略。

<Note>白名单费用配置可能适用于特定地址。因此，这些地址的代付行为可能与此处设置的付款偏好不同；调整某个地址的白名单归属可能会改变其适用的代付行为。</Note>

## TRON 手续费支付

对于 TRON USDT 转账，Fee Station 通过以下两种方式之一支付链上手续费：

- **能量模式（Energy mode）**：Fee Station 使用发送地址上可用或已租赁的能量来完成转账，从而避免燃烧 TRX，成本更低。在可以使用能量时，Fee Station 会优先采用该模式。
- **TRX 燃烧模式（TRX burn mode）**：当转账无法使用能量时，Fee Station 通过燃烧 TRX 支付手续费，成本高于能量模式。

TRON 转账显示的手续费反映实际采用的支付方式，因此不同转账显示的金额可能会因是否使用能量而不同。

<Note>如果您已在某个 TRON 发送地址上预先租赁了能量，但仍遇到 TRON 能量不足或手续费不足的报错，请确认 Fee Station 已为 TRON 启用并充值，且已在 **Gas Token** > **付款偏好** 中配置 TRON 的付款偏好。如果在 Fee Station 余额充足后报错仍未消除，则该地址可能需要完成额外的账户配置后才能使用预租赁的能量。</Note>
