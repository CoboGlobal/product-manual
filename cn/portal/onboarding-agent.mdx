## Cobo Portal Onboarding Agent 是什么？

Cobo Portal Onboarding Agent 是嵌入于 Cobo Portal 新用户 Onboarding 流程中的 AI 引导与协调助手，辅助用户完成从初始配置到首次交易的完整流程。

Onboarding Agent 会持续感知您当前的配置进度与系统状态，提供清晰的步骤指引、进度反馈与必要解释，帮助您在正确的顺序中完成关键配置，获得更简单、流畅的 Onboarding体验。

Onboarding Agent 不替您做出决策，也不会自动推进关键流程，而是作为一个随时可用的引导者，在您需要时提供明确的下一步建议和背景说明。

## 适用对象与使用场景

### 适用对象

- 新创建团队的管理员  
- 首次使用 Cobo Portal 的用户
- 需要重新理解或补全关键配置的团队成员

### 典型使用场景

- 首次进入 Cobo Portal，尚未完成基础配置  
- 需要完成安全相关的初始设置（例如 Cobo Guard 绑定）  
- 团队尚未邀请成员或未完成角色分配
- 不确定当前状态下可以执行哪些操作，或下一步应该做什么  

## 可协助完成的任务

Onboarding Agent 可调用多个下属 AI Agents 协同工作，帮助您完成完整的 Cobo Portal Onboarding。AI Agent 会根据当前任务，按需自动协作和切换。

<Info>
**什么是 Agent？**

与通用 AI 聊天工具不同，AI Agent 深度嵌入产品流程，面向特定任务场景，在用户授权前提下提供引导和发起操作。
</Info>

在使用过程中，Onboarding Agent 可以协助您完成以下任务：

- 结合产品文档内容，解答与 Cobo 产品和配置流程相关的常见问题  
- 引导 MPC 钱包新用户完成新手指引配置，包括：  
  - 下载和绑定 Cobo Guard
  - 邀请团队内成员并分配用户角色  
  - 配置 MPC 钱包，例如创建 Vault、创建主控私钥、创建和备份私钥分片、创建钱包等
- 解释团队内风控规则配置的含义
- 引导您体验资产充值与提币流程，并说明相关操作和注意事项

## 使用方式与交互流程

[注册登录 Cobo Portal](/cn/portal/sign-up) 后，您可从 Cobo Portal 左下角进入 AI Assistant (Onboarding Agent) 的对话窗口。

Onboarding Agent 以**对话式、分阶段**的方式与您交互，根据您当前所处的流程阶段和系统状态，提供相应的引导、说明与提示，协助完成操作。
