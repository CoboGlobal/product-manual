## Cobo Portal Onboarding Agents 是什么？

Onboarding Agents 是 [Cobo AI 助手](/cn/ai/introduction#cobo-ai-assistant) 在新用户接入阶段调用的一组协同工作的 Agents，可以持续感知您当前的配置进度与系统状态，提供清晰的步骤指引、进度反馈与必要解释，帮助您顺畅完成关键接入步骤。

Onboarding Agents 会为您提供合适的步骤引导，但不会自动推进关键流程，而是作为一个随时可用的接入助手。

<Info>
**什么是 Agent？**  

Agent 可以理解为 AI 助手在特定任务场景下调用的智能工具，不同 Agent 负责不同类型的任务。这些 Agents 会在对话中按需切换和协作，由 Cobo AI 助手统一协调，您无需手动选择或切换。
</Info>

## 适用对象与使用场景

Onboarding Agents 适用于通过自助注册方式创建的 Cobo Portal 免费试用团队，团队用户尚未完成新用户接入流程，且需要使用 MPC 钱包。

典型适用对象包括：

- 新创建团队的管理员，需要配置钱包、邀请成员等
- 首次使用 Cobo Portal 的团队成员，需要完成 Cobo Guard 等安全相关配置  
- 需要重新理解或补全关键配置的团队成员
- 不确定当前状态下可以执行哪些操作，或下一步应该做什么的用户

## 可协助完成的任务

Onboarding Agents 由多个 Agents 协同工作，可协助您完成以下任务：

- 结合产品文档内容，解答与 Cobo 产品和配置流程相关的常见问题  
- 引导 MPC 钱包新用户完成新手指引配置，包括：  
  - 下载和绑定 Cobo Guard  
  - 邀请团队内成员并分配用户角色  
  - 配置 MPC 钱包，例如创建 Vault、创建主控私钥、创建和备份私钥分片、创建钱包等  
- 解释团队内风控规则配置的含义  
- 引导您体验资产充值与提币流程，并说明相关操作和注意事项

<Info>
在对话的过程中，您可能会看到不同名称的 Agents 被调用。这表示当前由对应的 Agent 负责处理特定任务，例如 Member Invitation Agent 负责完成团队成员邀请，Cobo Guard Setup Agent 负责引导 Cobo Guard 设置等。
</Info>

## 使用方式与交互流程

[注册登录 Cobo Portal](/cn/portal/sign-up) 后，您可从 Cobo Portal 左下角进入 **Cobo AI 助手** 的对话窗口。

Cobo AI 助手将调用 Onboarding Agents，以**对话式、分阶段**的方式与您交互，根据您当前所处的流程阶段和系统状态，提供相应的引导、说明与提示，协助完成配置与操作。
