点击相应的按钮或图标，即可在开发者控制台中管理 API Key：

- **注册 API Key**：注册具有预定义用户角色、权限和 IP 白名单的 API Key，与 Cobo Portal 进行经过授权的交互。
- **编辑** <img src="/cn/images/developer-console/icon_edit.svg" className="icon"></img>：修改所选 API Key 的相关字段。
- **失效** <img src="/cn/images/developer-console/icon_revoke.svg" className="icon"></img>：永久禁用 Cobo Portal 与所选 API Key 之间的所有通信。
- **筛选** <img src="/cn/images/developer-console/icon_filter.svg" className="icon"></img>：根据 API Key 的状态和类型定制视图。
- **搜索** <img src="/cn/images/developer-console/icon_search.svg" className="icon"></img>：搜索 API Key 名称或相关关键字，快速查找 API Key。
