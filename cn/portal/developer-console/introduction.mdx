Cobo Portal 的开发者控制台是开发者统一管理 API Key、跟踪 API 日志、监督 Webhook 事件和处理 Callback 消息的平台。

开发者控制台分为 **WaaS 2.0** 和 **Cobo Portal Apps** 两个板块，提供了针对性的的工具和实时洞察，帮助开发人员简化审批流程，更轻松地管理 Cobo Portal Apps。**WaaS 2.0** 标签页提供了一系列 API 管理功能，而 **Cobo Portal Apps** 标签页则提供 Cobo Portal Apps 相关的监控功能。

开发者控制台提供了详尽的日志和实时指标，让开发人员访问所需的数据和资源，随时保持对团队使用 API 情况的全面掌控。
