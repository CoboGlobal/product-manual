## 前提条件

您已被分配了操作员、经理或管理员[用户角色](/cn/portal/organization/roles-and-permissions)。

## 步骤

1. 登录 [Cobo Portal](https://portal.cobo.com/login)。
2. 点击 <img src="/cn/images/mpc-wallet/wallets-icon.svg" className="icon"></img>> **MPC 钱包**。
3. 选择在[创建 MPC Vault](/cn/portal/mpc-wallets/ocw/create-mpc-vault) 步骤中创建的 vault。
4. 点击**创建钱包**。
5. 为钱包命名，然后点击**创建**。
<img src="/cn/images/mpc-wallet/create-mpc-wallet.png" className="screenshot_full_screen"/>

现在您已经创建了一个 MPC 钱包。

## 下一步

创建钱包后，您可以：

- 钱包
    - 查看钱包的交易历史。
    - 编辑钱包名称。

- 代币
    - 添加代币类型到您的钱包。如果您找不到想要的代币，可以[请求自列代币](/cn/portal/self-service-token-listing)。
    - 充币到钱包。
    - 从钱包提币。
    - 查看与代币相关的交易历史。
    - 查看代币的充币地址。
    - 搜索、过滤和排序钱包中的代币。
    <Note>MPC 钱包在所有支持的 EVM 链上使用相同的地址，您只需创建一次即可。已经生成的 EVM 链地址将自动在所有其他支持的 EVM 链上生效。
    目前，不同接口在查询 EVM 链地址时会有差异：
    - [API](https://www.cobo.com/developers/v2/api-references/wallets/list-wallet-addresses)：查询结果会受 `chain_id` 限制，只返回该链上的地址。  
    - Cobo Portal：会展示所有支持的 EVM 链上的地址，因此显示的地址数量可能比 API 返回的更多。</Note>

- 地址
    - 生成一个有唯一性的钱包地址。
    - 充币到地址。
    - 从地址提币。
    - 查看地址的交易历史。
    - 编辑地址标签。
    - 搜索、过滤和排序钱包中的地址。
