sidebarTitle: "TSS Node 版本说明"

版本 0.9.4:

- 支持 Schnorr 签名
- 更新 tss-node 脚本
- 支持密码分段功能，修改新密码长度限制
- 修改创建文件夹与文件权限

版本 0.9.0：
- 增加回调请求超时配置
- 增加回调服务监控配置
- 回调服务请求参数增加 tweaklist
- 回调服务配置支持 policy
- 修改了 export-share 导出 recovery group 文件格式
- 增加 change password 功能
- migrate 命令增加了部分数据的迁移与优化
- 安全审计的问题修复
- MPC 安全协议的优化与升级
- 支持连接 mysql 存储私钥
- 优化了系统整体性能与可靠性，修复了若干问题

版本 0.7.0：

- 增强了特定模块的安全性
- 增强了“migrate”命令，纳入数据迁移和优化
- 调整了配置文件和命令行的优先级
- 增加了在 callback 服务拒绝某项请求时通知用户的功能
- 提高了系统的整体性能和可靠性
- 修正了错误

版本 0.6.0：

- 改进了底层 MPC 协议的实现，以增强安全性
- 建立了与开发环境的默认连接
- 升级了系统消息格式和加密方法
- 重构了 TSS Node 注册机制
- 引入了规范组 ID 和协议组 ID
- 支持自动处理状态异常的会话
- 扩展了恢复文件数据字段
- 重构了风险控制 callback 机制
- 优化了数据库结构和日志系统
- 增强了系统整体性能和可靠性

版本 0.5.6：

- TSS Node 重新启动时将恢复未完成的会话
- 通过命令行查询公共信息时不再需要密码
- 修改了使用 export-share 命令导出的数据
- 修改了 API callback 参数中的请求详细信息
- 修改了文件中的日志显示
- 优化了系统可靠性和性能

版本 0.4.1：

- 代码重构
- 添加了对 Ed25519 签名机制的支持
- 支持“sign”子命令，用于定期检查 MPC 私钥分片
- TSS Node 组数据的数据库迁移
- 性能优化和错误修复

版本 0.3.0：

- 组信息可显示椭圆曲线名称、参与者信息、创建时间等信息
- 支持在数据库加密过程中随机生成加密盐
- 添加了对 Ed25519 签名机制的支持
- 优化了“export-share”子命令 
- 优化了 TSS Node 脚本文件
- 性能优化和错误修复

版本 0.2.7：

- 组信息可显示链码、私钥分片 ID、与 MPC 私钥分片关联的公钥信息等内容
- 支持 “export-share”子命令 
- 支持容器镜像打包和发布
- 性能优化和错误修复
