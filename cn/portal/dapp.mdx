当前，你可以通过以下两种方式让 Cobo 钱包与 dApp 交互：

| 交互方式             | 参考文档                                                                                           | 支持的链       | 支持的钱包类型               |
| -------------------- | -------------------------------------------------------------------------------------------------- | -------------- | ------------------------ |
| 通过 Cobo WaaS 2.0 API | [通过调用智能合约接口与 dApp 交互](https://www.cobo.com/developers/v2/api-references/transactions/call-smart-contract) | EVM、Solana    | 全托管钱包（Web3 钱包）、MPC 钱包     |
| 通过 Cobo Connect 插件 | [通过 Cobo Connect 与 dApp 交互](/cn/cobo-connect/interact-with-dapps)     | EVM            | 全托管钱包（Web3 钱包）、MPC 钱包     |
