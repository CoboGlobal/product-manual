**适用范围：所有钱包**

Cobo Portal 与 WaaS 2.0 API 均提供了开发环境（Developer Environment）与生产环境（Production Environment）。建议您先在开发环境中测试新功能，然后再将其部署到生产环境中。

如果您遇到找不到注册入口、激活链接打不开，或难以区分开发环境与生产环境的登录入口等问题，可参考以下说明：

1. 区分开发环境 URL
    - 开发环境 Cobo Portal URL：https://portal.dev.cobo.com/
    - 开发环境 WaaS 2.0 API Base URL：https://api.dev.cobo.com/
    - 生产环境 Cobo Portal URL：https://portal.cobo.com/
    - 生产环境 WaaS 2.0 API Base URL：https://api.cobo.com/

2. 账号与环境不互通
    - 开发环境与生产环境是不同环境，账号与组织数据通常不互通。请确保您使用的登录入口与邀请/开通的环境一致。

3. 未收到或需要重发激活邮件
    - 建议先检查垃圾邮件、企业邮箱拦截规则，以及是否使用了正确的邮箱地址。
    - 如仍无法激活，请联系 Cobo 客服（help@cobo.com）协助处理，并提供邮箱地址与组织名称。
