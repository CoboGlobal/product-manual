要访问套餐方案，请点击右上角的个人头像，然后选择**套餐方案**。

套餐方案页面包含以下选项卡：

- **套餐方案**: 查看和管理当前套餐方案，比较可用方案，并根据需要升级或降级。
- **账单**: 查看账单历史记录，下载发票，并跟踪付款状态。
- **优惠券**: 查看和管理应用于您账户的促销优惠券或积分。

Cobo 提供多种付费套餐，满足您多样的业务需求。您可在官网的[套餐介绍页](https://www.cobo.com/zh/pricing)查询每个套餐的具体内容。每个套餐都包含既定的用量配额，例如可以生成的地址数量、可以使用的链数量，以及可以在 Cobo 服务下存储的资产总量等。

## 套餐选择

- **入门版和标准版套餐**：入门版和标准版套餐专门面向使用需求相对较低的小型企业和成长型企业。
- **专业版套餐**：专业版套餐专门面向使用需求较高和需要定制化的企业。如果对我们的专业版套餐感兴趣，[请联系我们的销售团队](https://www.cobo.com/book-demo/?utm_source=product-manual&utm_medium=product&utm_campaign=pricing-plan-intro&utm_content=enterprise-contact-sales)，他们将协助您找到最符合您需求的定制套餐。

专业版套餐提供两种不同的用量包配额类别，分别用于满足 WaaS 客户和财资管理客户的特定需求。下表概述了每个配额类别的主要特点：


|      | 目标用户                                | 示例                         | 特点               |
| ---- | ----------------------------------- | -------------------------- | ---------------- |
| WaaS | 使用 Cobo 的 WaaS API 以编程方式创建数字资产钱包的企业 | 交易所、加密原生公司、OTC 和券商以及零售平台   | 提币金额和钱包地址数量的额度较高 |
| 财资管理 | 希望直接在 Cobo Portal 上保护和管理其数字资产的企业    | DeFi 和质押基金、交易公司、机构投资方和金融服务 | 资产托管金额的额度较高      |


- **选择套餐的决策树**：
  - **第一步：选择钱包类型**：您首先应该考虑您是希望使用全托管钱包还是MPC钱包
    - 入门版和标准版套餐仅提供MPC钱包
    - 专业版钱包则同时支持全托管钱包和MPC钱包。


    - **如何选择全托管或MPC钱包？**
    - **1. 考虑监管或客户要求**：监管方或客户可能对您使用持牌的机构级托管或是有多方签名的机制有要求，这将决定您必须使用全托管钱包还是MPC钱包，两种场景Cobo都是满足需求的服务机构，但入门版和标准版套餐仅提供MPC钱包，而专业版钱包则同时支持全托管钱包和MPC钱包。
    - **2. 考虑KYT**：考虑您是否需要以及如何做KYT。全托管钱包将自动为您做KYT，KYT费用包含在交易费用里。MPC钱包而不包含自动的KYT，您可以选择Cobo的ScreeningApp帮助您做KYT，或自己完成KYT。如果您不需要做KYT，则MPC钱包更适合您。
    - **3. 考虑集成难度**：全托管钱包集成相对简单，适合寻求易用性和便捷性的客户，因为它减少了客户在安全架构和私钥管理方面的负担。MPC 钱包集成相对复杂一些，您可能需要将私钥分片和签名逻辑集成到您的 App 中（例如 2-2 模式下，终端用户持有其中一个分片）。因此如果您希望简单快速便捷地使用Cobo服务，可以优先选择全托管钱包。
  - **第二步：根据用量选择**：
    - 如果您需要全托管钱包，则需要专业版。
    - 如果您需要MPC钱包，您可以依次从以下用量来考虑选择来考虑选择入门版、标准版、专业版。
      - 月度提币金额
      - 钱包地址数
      - API调用次数
      - 用户成员数
    - Tips：如果您的项目在起步阶段，可以选择用量较低的套餐，当项目成长后，您可以选择支付少量超量费用，或随时升级到更高等级的套餐。

## 使用项目

下表概述了付费套餐中包含的使用项目。请注意，具体的使用项目可能因您选择的套餐而异。

- **累计提币金额**：您提取到外部地址的资产总量。全托管钱包、MPC 钱包和智能合约钱包内部和之间的转账不计入累计提币金额。
- **资产托管金额**：您存储在 Cobo Portal 中的资产总量。
- **钱包地址数**：您生成的地址数量。
- **可用公链数**：您已启用的链的数量。链一旦启用，即视为正在使用，并将计入套餐使用量。如需启用或禁用公链，请参阅[启用或禁用公链](/cn/portal/enable-or-disable-chains)。
- **API 调用次数**：您发送的 WaaS API 请求数量。该数量在整个团队范围内计算。
- **用户成员数**：您所在团队的用户数量。该数量在整个团队范围内计算。

使用**优惠券**选项卡可以查看和管理应用于您账户的促销优惠券或积分。

对于 14 天免费试用，一旦试用期结束或达到用量限制，您将只能浏览您的 MPC 钱包和交易所钱包。您必须购买付费套餐才可重新使用完整功能。
