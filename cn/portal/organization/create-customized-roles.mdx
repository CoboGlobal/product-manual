除了 Cobo 提供的预设用户角色外，您还可以创建自己的定制用户角色。

要创建新用户角色，请点击**团队管理** > **用户角色** > **新建角色**。

<img src="/cn/images/organization/customize.png" className="screenshot_modal"/>

**注意**：在**新建角色**对话框中，请指定此用户角色将应用于 Web 操作还是 API 操作。
